My HTML & CSS CheatSheet

HTML -HyperText Markup Language- Table References

Element Description
<head> Is a container for metada like title, styles, scripts, and other meta info
<h1> Is the main "title"/header of the page.
<table> To create a table, will contain th(header), tr(row), td(cell data)

CSS -Cascading Style Sheet- Table References

Element Description
text-align Will align the text to the left, right or center
background-color Will set the background color of the "element" to the selected color
font-family Will set the font of the "element" to the selected font

Git Table References

Element Description
git status Show git status on the current branch
git checkout -b To create a new branch (Name should be entered after -b)
git log Show git previous logs
